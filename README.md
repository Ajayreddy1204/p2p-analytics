# DealerPulse - P2P Analytics Dashboard

## Overview
DealerPulse is a comprehensive P2P (Procure-to-Pay) analytics dashboard for dealer performance management. It provides real-time insights into dealer KPIs, AI-powered analytics, and predictive forecasting.

## Architecture
- **Frontend**: Streamlit
- **Backend**: Python
- **Database**: Redshift / Aurora PostgreSQL
- **AI Services**: AWS Bedrock, Amazon Q
- **Deployment**: AWS EC2

## Features
- Real-time KPI tracking and monitoring
- AI-powered natural language queries
- Predictive analytics and forecasting
- Dealer health scorecards
- Transaction lineage tracking
- Strategic insights generation
